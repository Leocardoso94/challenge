## Setup
Instale as dependências do projeto usando:
```
yarn install 
// ou
npm install
```

## Rodando o projeto

```
npm start
```

Este comando irá gerar os arquivos dentro da pasta public e subir o servidor, agora basta acessar a url http://localhost:8080/
